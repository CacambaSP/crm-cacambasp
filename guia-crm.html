➕ Como Abrir uma Nova Ordem
1

Envie o link de cadastro ao cliente

Antes de abrir manualmente, prefira enviar o link cacambasp-crm.vercel.app/cadastro.html pelo WhatsApp. O cliente preenche os dados e o card é criado automaticamente.

2

Ou clique em "+ Nova Ordem" no topo

Preencha: nome, celular, endereço, bairro, CEP, tipo de serviço, data e valor cobrado. O CEP preenche o endereço automaticamente.

3

Atribua o fornecedor

Clique no card → Editar → escolha o Fornecedor → escolha a Região (o custo preenche automaticamente com o valor tabelado).

💡 O sistema sugere o fornecedor que mais atendeu aquele bairro!
4

Envie a OS ao cliente

Clique no card → botão OS → 📎 PDF + WhatsApp. O PDF é gerado e o WhatsApp abre com a mensagem já preenchida.

🏆 Como Concluir uma Ordem
1

Confirme que o cliente pagou

Clique no card → Cliente pagou → informe o valor e a data do recebimento.

2

Clique no 🏆 no card

Passe o mouse sobre o card no funil → aparece o ícone 🏆 no canto inferior direito → clique para mover para o Histórico.

3

Consulte no Histórico

Menu → Histórico → filtre por período para ver os totais de Cobrado, Custo e Margem.

⚠️ Alertas Importantes
!

Alerta de vencimento (3 dias)

Ao entrar no sistema, se houver clientes com caçamba há 3 dias ou mais, aparece um aviso laranja. Clique nele para enviar mensagem de renovação pelo WhatsApp automaticamente.

⚠️ Nunca exclua um card sem ter certeza. A exclusão é permanente e não pode ser desfeita.
